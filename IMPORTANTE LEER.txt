Antes de trabajar en el archivo se debe realizar el siguiente proceso:

1) Descargar e instalar el setup global de composer (Si ya lo tienes instalado omitir este paso). 

Link: https://getcomposer.org/download/

2) Abrir la terminal GIT BASH con la dirección de la carpeta telecomunicate (Otras terminales no permiten usar el comando cp)

3) colocar en la terminal el siguiente comando:

composer install / composer update (Cualquiera funciona SOLO USAR UNO) 

4) Se instalará la carpeta vendor con todas las dependencias y una carpeta public con los archivos globales del sistema (JQuery, fontawesome, bootstrap)
